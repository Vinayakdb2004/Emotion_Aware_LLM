# Emotion_Aware_LLM
EmpathAI — Emotion-Driven Response Generation System 2025 Conversational AI  

• Emotion-aware AI using BERT-base + LoRA for high-accuracy multiclass emotion classification across customer queries.
• LLaMA-3 response generation fine-tuned on EmpatheticDialogues/DailyDialog for context-aware, empathetic replies.
• System auto-detects sentiment categories (anger, frustration, joy, confusion) and generates tailored supportive responses.
